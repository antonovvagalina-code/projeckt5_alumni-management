Alumni Connect - это многофункциональная система управления сообществом выпускников, разработанная на языке Java с использованием библиотеки Swing для создания надежного и удобного интерфейса.

Архитектура приложения построена на основе CardLayout, что обеспечивает плавную и интуитивную навигацию между всеми функциональными модулями. При запуске пользователя встречает приветственный экран с возможностью авторизации для входа в систему.

Главное меню предоставляет доступ к четырем ключевым разделам:
- Модуль добавления выпускников содержит форму с полями для внесения полной информации: ФИО, год выпуска, факультет, электронная почта, телефон и должность. Предусмотрена функция загрузки фотографии и сохранения данных.
- Модуль просмотра отображает список всех выпускников в удобном формате с возможностью прокрутки.
- Галерея событий позволяет просматривать фотографии с мероприятий, создавая визуальную историю жизни сообщества.

Система обеспечивает централизованное хранение всех данных о выпускниках, быстрый доступ к информации и профессиональный пользовательский интерфейс. Alumni Connect - это готовое решение для образовательных учреждений, стремящихся поддерживать прочные связи со своими выпускниками.
